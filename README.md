2. Crea una clase llamada Asiento que represente los asientos de una salade cine o teatro.
   
La clase tendrá 4 atributos de distintos tipos:

 int fila

 int columna

 boolean ocupado

 double precio

La clase incluirá tres métodos:

a) mostrarInfo(): muestra una línea con la fila, la columna, el precio y un
símbolo "X" si está ocupado o "O" si está libre.

b) ocupar(): cambia el estado del asiento a ocupado.

c) main(): crea una matriz de 3×4 asientos, asigna valores aleatorios y
muestra el estado completo de la sala en formato visual.
